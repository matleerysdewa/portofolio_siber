Bruteforce Semua Shift

cipher = "ynkooejcpdanqxeykjrbhqkrv"
for shift in range(26):
    plain = "".join(
        chr((ord(c) - ord('a') - shift) % 26 + ord('a'))
        for c in cipher
    )
    print(f"Shift {shift:2d}: {plain}")

# Output shift 4:
# crossingtherubicondjneoach

1

Analisis Website

Website menampilkan string random dan meminta kita submit MD5 hash-nya. Form expire dalam hitungan milidetik — tidak bisa dilakukan manual.

2

Automasi dengan Python requests

import requests, hashlib
from bs4 import BeautifulSoup

s = requests.Session()
r = s.get("http://TARGET/")
soup = BeautifulSoup(r.text, 'html.parser')
string = soup.find('h3').text.strip()
md5 = hashlib.md5(string.encode()).hexdigest()
r2 = s.post("http://TARGET/", data={'hash': md5})
print(r2.text)
